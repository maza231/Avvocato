# Project Rules

This project is a modern website for a lawyer.

## Tech stack
- React / Next.js
- Tailwind CSS
- Mobile-first design
- Clean component structure

## Design style
- Modern, premium, simple
- Strong hero section
- Clear CTA buttons
- Good spacing
- No generic AI-looking layout
- No random gradients unless useful

## Code rules
- Use reusable components
- Keep code readable
- Avoid unnecessary dependencies
- Make all pages responsive
- Optimize images
- Test layout on mobile

## Business goal
This is a demo website for a lawyer whose goal is to attract more clients and showcase the services they offer. Get the necessary information from this website: https://www.compagno.biz/
